# ShambaPulse AI 🌾

**ShambaPulse AI** is a warm, empathetic, and savvy AI Market Negotiator designed specifically for Kenyan farmers. It helps farmers protect their livelihoods by providing predictive pricing, market insights, and strategic negotiation tactics—fluidly switching between **Sheng** and **English** to feel like a trusted "Rafiki" (friend).

## 🚀 Features

- **Strategic AI Negotiator:** Fluid code-switching (Sheng/English) AI agent that helps farmers handle brokers and find the best time to sell.
- **Market Pulse Dashboard:** Real-time predictive trends for major commodities like Maize, Beans, and Tomatoes.
- **Historical Analytics:** Visual price history charts to identify seasonal patterns and market shifts.
- **Livelihood Protection:** Intelligent "Negotiator's Tips" that alert farmers to regional price peaks (e.g., supply tightening in Thika or Nakuru).
- **Fully Responsive:** Beautifully crafted "Editorial Aesthetic" that works seamlessly from narrow mobile screens to large desktop monitors.

## 🛠️ Tech Stack

- **Frontend:** React 19, TypeScript, Vite
- **AI Intelligence:** Google Gemini (via `@google/genai`)
- **Animations:** Framer Motion (motion/react)
- **Styling:** Tailwind CSS (Modern v4 style)
- **Data Visualization:** Recharts
- **Icons:** Lucide React

## 📦 Getting Started

### Prerequisites

- Node.js (v18+)
- A [Google AI Studio API Key](https://aistudio.google.com/app/apikey)

### Installation

1. **Clone the project** (if exported) or continue in the AI Studio environment.
2. **Install dependencies:**
   ```bash
   npm install
   ```
3. **Configure Environment Variables:**
   Create a `.env` file in the root directory and add your Gemini API key:
   ```env
   GEMINI_API_KEY="your_api_key_here"
   ```
4. **Start the Development Server:**
   ```bash
   npm run dev
   ```

## 🎨 Design Philosophy

The application utilizes an **Editorial Aesthetic**, characterized by:
- **Typography:** Sophisticated serif headers (*Playfair Display*) paired with clean functional sans-serif body text (*Inter*).
- **Color Palette:** "Earth & Strategy" — muted stone tones, heritage greens (`#526D4E`), and terracotta accents (`#D48D6A`).
- **Layout:** High-contrast bento-grid structures for data insights and a conversational-first interface for the AI agent.
